# 学习历程

1.数学基础：
基本形式：y=θ0+θ1x+θ2x+...+θnx
代价函数：J(θ)=1/2m*∑(y_pred-y)^2
正规方程（无正则化）：θ=(X^T*X)^-1*X^T*Y
L2正则化正规方程：θ=(X^T*X+λI)^-1*X^T*Y
梯度下降（无正则化）：偏导J(θ)=（1/m）*X^T*(X*θ-y)
梯度下降（L2正则化）：偏导J(θ)=（1/m）*X^T*(X*θ-y)+λ*θ
参数更新：θ=θ-α*偏导J(θ)

2.编程基础：
熟悉从sklearn导入数据，计算代价函数、梯度下降、参数更新的逻辑，划分训练集和测试集，可视化训练过程

# 对结果的分析和猜想

一、linear_regression_no_reg
1.Gradient Descent Convergence Curve (No Regularization)，梯度下降曲线前期变化快，后期变化慢，模型拟合度高。
2.True vs Predicted (Test Set)，预测值和真实值拟合直线斜率近似1且点集中于直线附近，模型拟合度高。
3.Residual Plot (Test Set)，残差分布图基本在-2~2之间，模型拟合度高。

二、linear_regression_with_reg
1.λ= 100.0  | Train R²=0.6119 | Test R²=0.5780 | ||θ||₂=1.5007，此结果为最优结果，因此选取λ=100.0作为正则化参数。
2.λ vs R² Score和λ vs MSE训练集远优于测试集，过拟合严重。
3.随λ变化，λ vs R² Score和λ vs MSE变化不大，正则化对过拟合的影响有限，λ=100.0时，稍微缓解了过拟合。
4.λ vs Weight Shrinkage，只有λ=100.0时显著下降，说明较小的λ对模型参数的正则化效果较小。
5.【效果对比】   无正则化 - Test R²: 0.5758   Ridge(λ=100.0) - Test R²: 0.5780   最终结果添加正则化会使拟合效果稍微提升。

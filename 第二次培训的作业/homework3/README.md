# 结果分析和猜测

1.用sebal分类发现效果差，用petal分类发现效果好。
2.用linear核函数发现效果好,准确率达0.9111，用rbf核函数发现效果差。
3.可视化当中，setosa与另外两类分的比较开，另外两类有部分重叠。因此在二维特征上的决策边界可视化发现，setosa预测全对，而另外两类有错误。
4.最后sepal分类效果也不好，因此选择petal分类。

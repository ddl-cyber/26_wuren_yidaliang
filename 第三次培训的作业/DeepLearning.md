# 深度学习笔记
## 概念的深入研究
### 反向传播算法/梯度下降算法
#### 详细版原理
1. 前向传播：根据现有的参数，经过输入层、隐藏层、激活函数、输出层，计算出输出层的输出。
2. 链式法则对输入求偏导：偏LOSS偏当前层输入=偏LOSS偏当前层的输出*偏当前层的激活函数偏当前层输入
3. 链式法则对参数求偏导：偏LOSS偏当前层参数=偏LOSS偏下一层的输入*偏下一层的输入偏当前层参数（当前层的输出）
4. 链式法则对输出层求偏导：偏LOSS偏输出层的输出=偏LOSS偏下一层的输入*偏下一层的输入偏输出层的输出（当前层的参数），最终输出是y预测，LOSS=（1/2）（y预测-y真实）^2是迭代终点。（其实3和4是对称的）
5. 更新参数：参数更新为->参数-学习率*偏LOSS偏参数
6. 重复：LOSS收敛到最小
### 正则化
#### 基本原理：损失函数=原损失函数+λ*惩罚函数
- L1正则化：惩罚函数=λ*sum(|参数|)
- L2正则化：惩罚函数=0.5*λ*sum(参数^2)
#### 结构化正则化与稀疏学习
- Group Lasso（组稀疏）：将参数按组进行惩罚，实现整组参数的同时稀疏化，适用于结构化特征选择和多任务学习。
- Structured Sparsity：在模型压缩（Pruning）中，通过结构化稀疏正则化可以裁剪整个神经元、通道甚至层，实现硬件友好的模型加速。
#### 自适应正则化（Adaptive Regularization）
- AutoML + 贝叶斯先验：通过自动机器学习技术动态调整正则化强度λ，使正则化系数随训练过程和不同层的需求自适应变化。
- 可微分正则化：将正则化策略本身作为可学习参数，通过梯度下降联合优化，如 Meta-Learning 中的自适应权重衰减。
#### 对抗性正则化（Adversarial Regularization）
- 对抗训练（Adversarial Training）：在训练数据中引入对抗性扰动，迫使模型在输入空间的局部邻域内保持稳定，提升模型对抗攻击的防御能力和整体鲁棒性。
- 虚拟对抗训练（VAT）：无需标签信息，通过最小化输入扰动前后的输出分布差异实现半监督正则化。
#### 新型 Dropout 变体（原体是随机丢弃神经元）
- DropConnect：不是随机丢弃神经元激活值，而是随机将权重连接置零，比标准 Dropout 更细粒度。
- ShakeDrop / Shake-Shake：在训练时对分支输出施加随机缩放扰动，有效增强残差网络等深层架构的正则化效果 3。
- Spatial Dropout：在卷积网络中按通道维度整张特征图丢弃，适用于图像语义分割等任务

## 课外拓展
### CNN以外的其他网络结构
- 多层感知机（MLP）：全连接
- 循环神经网络（RNN）：专门处理序列数据，网络有"记忆"，能记住前面的信息影响后面的输出。LSTM（长短期记忆网络）：解决RNN"健忘"问题，能记住长期依赖；GRU：LSTM的简化版，参数更少。
- 生成对抗网络（GAN）：可用于数据增强。结构：两个网络互相博弈。生成器（Generator）：造假（生成假图片）。判别器（Discriminator）：鉴真（判断真假）。两者**协同进化**
- 注意力机制网络（Transformer）：在序列数据处理中，通过自注意力机制，关注序列中的不同位置之间的关系，提高模型对序列信息的理解能力。查询+索引+内容。多头注意力并行
- 强化学习网络（DQN）：状态+动作+奖励+策略
### 梯度下降算法以外的其他优化算法
- 牛顿法（Newton's Method）：通过计算二阶导数（Hessian），直接求解损失函数的最小值点，比梯度下降更快。
- 进化算法（Evolutionary Algorithms）：模拟自然选择和遗传机制（增加随机噪声，如同变异），通过迭代搜索解空间，找到全局最优解。包括遗传算法（GAetic Algorithm）、粒子群优化（Particle Swarm Optimization）等。
### Transforms变换以外的其他数据增强技术
#### 输入图像的数据增强技术
- 随机遮挡（Random Erasing）：在训练时随机遮挡图像的一部分，增加模型的泛化能力。
- 图像混合（Mixup）：在训练时随机将两个图像混合，生成新的图像，增加数据的多样性。
- CutMix：在训练时随机裁剪图像的一部分，将裁剪部分与其他图像混合，生成新的图像，增加数据的多样性。
- Mosaic：在训练时随机将图像的多个小块与其他图像混合，生成新的图像，增加数据的多样性。
- 复制粘贴（Copy-Paste）：在训练时随机将图像的一部分复制粘贴到其他位置，生成新的图像，增加数据的多样性。
- GAN数据增强：利用GAN生成新的图像，增加数据的多样性。
- Diffusion模型：通过生成图像的扩散过程，将图像的生成任务分解为多个步骤，每个步骤生成图像的一个部分，最终生成完整的图像。
- 神经风格迁移（Neural Style Transfer）：将图像的风格（如画风、颜色等）应用到另一张图像上，生成新的图像。
- 自动增强（AutoAugment/RandAugment/TrivialAugment）：通过自动搜索最优的增强操作组合，增加数据的多样性，提高模型的泛化能力。
#### 网络结构里面的数据增强
-  Manifold Mixup：在训练时随机将两个样本混合，生成新的样本，增加数据的多样性。
- Feature CutMix：在训练时随机裁剪样本的一部分，将裁剪部分与其他样本混合，生成新的样本，增加数据的多样性。
- Sample Pairing：类似 CutMix，但混合比例由注意力图决定（模型认为重要的区域少被替换）。

## 本次任务后对深度学习的认识
1. 数据增强作为数据预处理的重要部分。选择搭建合适的神经网络结构并且选择合适的优化算法，是提高模型泛化能力的重要步骤。为了缓解过拟合问题，我们还可以使用正则化技术（如L1正则化、L2正则化、Dropout等）来限制模型的复杂度，提高模型的泛化能力。
2. 实操了神经网络的搭建和训练，对神经网络的原理有了更深的理解，对模型泛化能力的提升方法有了更深入的认识。
3. 全局平均池化（GAP）的深层意义：GAP 强制让每个通道的特征图与类别建立直接对应关系，使得网络必须学习更有区分度的全局特征，而不是依赖全连接层去"记忆"空间位置信息。这种结构上的归纳偏置（inductive bias）天然赋予了网络平移不变性2，比单纯用 Dropout 来防止过拟合更加优雅。
4. 学习率调度：学习率衰减的本质是在训练初期用大步长快速探索，后期用小步长精细收敛。如果只用固定学习率，后期可能在最优解附近来回震荡而无法进一步下降。
5. 辩证放弃早停：早停的本质假设是"验证集 loss 不再下降 = 开始过拟合"，但对于一个本身就不易过拟合的轻量网络而言，验证准确率在震荡中上涨是正常的，提前截断反而浪费了潜力。
6. 通道数逐步翻倍思想：空间尺寸缩小一半，通道数翻一倍，保持计算量大致不变
7. 直观理解了训练阶段变化：训练三阶段：快速提升（1-5 epoch）、稳定提升（20-70 epoch）、收敛震荡（70-200 epoch）
